# Welcome to my new blog!

This is my very first blog post, and it's not an April's fool! I decided to create it to share thoughts, things I learn in my journey, projects I'm working on and honestly everything I wish to let the world see, without using my more "professional" voice (e.g. LinkedIn). 

## Technical Details
This blog doesn't use any framework, it's entirely **Vanilla JavaScript** running inside your browser to parse the Markdown files I write directly into standard HTML.

I also thought about using something like Substack, but it seemed like "too much" for what I think this blog is going to be, and I always wanted to create something similar from scratch, working multiple times with non-technical people that would've enjoyed being able to update the websites themselves without touching the code.

Well, I guess that's all for today...

Thanks for visiting!
